
Musanze Market Order System 
A professional management portal for tracking supplier registrations and market orders in Musanze. Optimized for mobile and desktop with a premium Rose Gold aesthetic.

Core Features
1. Live Dashboard
Displays total order counts and RWF values for the current day. Includes a quick-view table of the 5 most recent activities.

2. Supplier Management
A dedicated portal to register new vendors and link them to specific market orders for organized record-keeping.

3. Smart Order Entry
Features real-time JavaScript calculation of total amounts and touch-optimized forms that fit perfectly on phone screens.

4. Order History
A complete log of all transactions with formatted dates and currency. Tables include horizontal scrolling to prevent layout breaking on mobile.

5. Database Control
Includes a built-in reset function to clear records directly from the interface, protected by a safety confirmation prompt.

7. Technical Design
Theme: Rose Gold (#b76e79), Deep Maroon (#6d1b3b), and Soft Blush (#fff5f7).
